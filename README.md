# Calculadora do churrasco
## Intro
O app facilita a divisão dos gastos do churrasco, ou qualquer evento, entre os participantes. O usuário registrar todos os participantes, suas respectivas despesas no churrasco e também os pagamentos feitos entre os participantes e calcula automaticamente os totais de quem deve pagar quanto para quem para que as despesas fiquem divididas de forma justa.

O meu objetivo com o projeto foi criar um projeto em ReactJS componetizado para o bootcamp da DIO/Take Blip. A ideia de rateio do churrasco foi do [Lucas Santos](https://github.com/LucasASantos/ls-rateio-churras) da Take Blip, que propôs o projeto em uma live do bootcamp. Muito obrigado por todo o apoio, Lucas!

## Requisitos
-- Node.Js  
-- npm

## Para rodar o app
Na pasta do programa execute "npm i" pelo terminal para instalar as dependências e depois execute "npm start" para iniciar. Pode demorar uns minutos para abrir, é normal.
